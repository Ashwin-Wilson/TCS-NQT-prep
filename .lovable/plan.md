

## TCS NQT 2026 Information Website

### Page 1: Homepage (All-in-one overview)
- **Hero Section** — "TCS NQT 2026" title with exam date/tagline and a brief description of what the exam is about
- **WhatsApp CTA Card** — A prominent, clickable card saying "Click to join WhatsApp channel for preparation" that opens the user's WhatsApp link (to be provided). Styled as an eye-catching card near the top.
- **Exam Pattern Section** — Cards/table showing:
  - Sections: Verbal Ability, Reasoning, Numerical Ability, Coding
  - Number of questions per section
  - Marks distribution
  - Difficulty level indicators
- **Time Schedule Section** — Key dates, registration deadlines, exam windows, and slot timings displayed in a clean timeline or card layout
- **Question Timer Overview** — Brief summary of time limits per section, with a CTA button "Practice with Timer →" linking to the practice page

### Page 2: Practice Timer Page
- Section selector (choose which section to practice: Verbal, Reasoning, Numerical, Coding)
- Countdown timer that matches the actual exam time per section
- Start/Pause/Reset controls
- Visual progress indicator showing time remaining
- Tips section for the selected section

### Design
- Clean & minimal style with subtle blue/teal accents
- Responsive for mobile (students often browse on phones)
- Smooth scroll between sections on homepage
